# Text Encryptor
## Description
A simple code to encrypt and decrypt text that uses AES encryption algorithm

## Usage

### Installation
Install the dependencies
```bash
pip install -r requirements.txt
```
### Running
```bash
python3 fernet_cli.py
```
```bash
python3 fernet_pyautogui.py
```
